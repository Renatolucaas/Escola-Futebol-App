⚽ Escola Futebol App

Aplicativo Android para gestão de agenda e eventos de escolas de futebol</b><br> Desenvolvido em <b>Kotlin</b> com Firebase.
📱 Sobre o Projeto

O Escola Futebol App é uma solução mobile que permite o gerenciamento completo de atividades de uma escola de futebol, incluindo treinos, amistosos e eventos sociais.

Com interface intuitiva e integração em tempo real com Firebase, técnicos e coordenadores podem organizar toda a programação da instituição de forma simples, moderna e eficiente.

✨ Funcionalidades Principais

📅 Calendário interativo de eventos

⚽ Cadastro de treinos por categoria

🏆 Gestão de amistosos e jogos

🎉 Eventos sociais e confraternizações

🔔 Notificações e lembretes

📊 Relatórios de atividades
